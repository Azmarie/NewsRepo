in desktop (or other directory) create file "a.sh"
its path is
~/Desktop/a.sh

edit this "a.sh"
insert:
touch ~/Desktop/`date +%s`
(the above command will create a file)
save

!!! Must set "a.sh" to <excutable>


in terminal, type
crontab -e
edit the file, add the following line at bottom of file
*/1 * * * * ~/Desktop/a.sh
(means the scheduler will run the above script every 1 minutes)
